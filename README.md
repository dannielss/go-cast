## :gift: Features

- List all streams
- Start a live stream
- Watch a live stream
- Real time chat

## :wrench: Configuration

`Golang version: 1.24.1`

```
1. Start application
$ make run
```

## :rocket: Technologies

- Golang
- Gorilla
- WebRTC
- JavaScript
- TailwindCSS
